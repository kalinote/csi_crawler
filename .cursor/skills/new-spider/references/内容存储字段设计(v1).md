# 内容存储字段设计(v1)
这里是一些公共字段设计，所有数据存储都应该包含这些功能字段。

| **字段名** | **类型** | 来源  | **描述与用途** |
| --- | --- | --- | --- |
| **uuid** | `keyword` | 采集  | **主键**。`md5(entity_type + source_id + last_edit_at + raw_content)`。用于去重或版本控制。 |
| **source\_id** | `keyword` | 采集  | 平台原本的ID（如推文ID、文章ID）。不同类型的数据source\_id含义有可能不同，但source\_id一定能追踪到同一个数据源头。 |
| **data\_version** | `integer` | 采集  | 数据版本号，用于追踪后续可能出现的字段更新，比如在这一套字段设计中应固定为1 |
| **entity\_type** | `str` | 采集  | 实体类型，该字段主要用于各中间件分类。 |
| **url** | `keyword` | 采集  | 原始链接。使用 `keyword` 也就是为了能精确匹配查询。 |
| **tags** | `keyword` (Array) | 采集  | 从原文内容中提取出的标签（如 #AI）。【如果该平台存在标签】 |
| **platform** | `keyword` | 采集  | 来源平台 (twitter, weibo, bbc, etc.)。 |
| **section** | `keyword` | 采集  | 所属板块。 |
| **spider\_name** | `keyword` | 采集  | 爬虫名称。 |
| **update\_at** | `date` | 自动生成 | 最后一次存入数据库时间 |
| **crawled\_at** | `date` | 采集  | 采集入库的时间。 |
| **publish\_at** | `date` | 采集  | 内容在平台上发布的实际时间（不同于采集时间）。 |
| **last\_edit\_at** | `date` | 采集  | 内容最后编辑时间（不同于发表时间）。 |
| **clean\_content** | `text` | 分析  | 清理后的纯文本，用于全文检索（需配置分词器）。 |
| **clean\_content\_vector** | `dense_vector` | 分析  | 纯文本嵌入词向量，用于关联搜索 |
| **vector\_status** | `boolean` | 分析  | 纯文本嵌入词向量是否已生成，根据此字段来判断是否已经生成词向量，而不是根据是否存在词向量字段来判断，防止部分clean\_content内容为空的数据被反复处理 |
| **raw\_content** | `text` (no index) | 采集  | 原始JSON、XML或HTML等。设为 `index: false` 以节省空间，仅做存储展示，不参与搜索。 |
| **safe\_raw\_content** | `text` (no index) | 分析  | 经过处理js和css后的安全网页内容，用于前端页面还原展示，该字段由用户手动触发页面还原任务后填充 |
| **language** | `keyword` | 分析  | 语种 (ZHO, ENG)，用于分词器选择和过滤，格式为ISO 639-3 code。 |
| **author\_id** | `keyword` | 采集  | 作者在原平台的唯一ID。 |
| **author\_name** | `text` / `keyword` | 采集  | 作者昵称。保留 `text` 用于模糊搜，`keyword` 用于聚合统计。 |
| **nsfw** | `boolean` | 采集/分析 | 是否为NSFW内容。 |
| **aigc** | `boolean` | 采集/分析 | 是否为AI生成内容。 |
| **translate\_content** | `text` | 分析  | 基于 clean\_content 翻译到中文的内容。【如果原文不是中文】 |
| **keywords** | `keyword` (Array) | 分析  | 基于内容分析得到的关键词列表。 |
| **entities** | `dict[str, list[str]]` | 分析  | 基于内容分析得到的实体dict。 |
| **emotion** | `float` | 分析  | 基于内容分析的情感分数，从-1(消极)到1(积极) |
| **political\_bias** | `keyword` (Array) | 分析  | 基于内容分析的政治倾向 |
| **confidence** | `float` | 分析/输入 | 内容置信度，范围0-1，其中0为零信任模式 |
| **subjective\_rating** | `integer` | 输入  | 主观评分，仅能由用户设置。 |
| **is\_highlighted** | `boolean` | 输入  | 是否为重点目标。 |
| **highlighted\_at** | `date` | 输入  | 重点目标标记时间。【如果被标记为重点目标】 |
| **highlight\_reason** | `text` | 输入  | 重点目标标记理由。【如果被标记为重点目标】 |
